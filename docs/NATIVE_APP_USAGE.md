# ネイティブiOSアプリ 使い方ガイド

iPhone単体で動く「記憶を持つAI」の使い方です。**通信が無くても**会話できます。

---

## 1. はじめて開く
1. アプリを起動すると **「モデルを準備しています…」** と出ます。
2. **初回だけ**モデルをダウンロードします（約1.8GB・**Wi-Fi推奨**・数分）。
3. **「おかえりなさいませ。」** が出れば準備完了。2回目以降は即起動します（オフラインでも）。

## 2. 会話する
- 下の入力欄にテキストを入れて、**↑ボタン**で送信。
- 端末内のAIが**その場で（通信なしで）**考えて返事します。
- 少し待ち時間があります（端末内で計算しているため）。「考えています…」と表示されます。

## 3. 記憶を同期する（自分のメモを参照させる）
最初は「脳みそだけ」の状態で、あなたのメモを知りません。**記憶を同期**すると、メモに基づいて答えられるようになります。
1. 右上の **🔄（同期ボタン）** をタップ
2. **接続先URL** と **パスワード**（Web版のログインパスワード）を入力
3. **「記憶を同期する」** → 「✅ 同期完了：記憶 N件」と出ればOK
4. ヘッダーが「記憶 N件・オフライン」になります

> 一度同期すれば、**機内モードでも**その記憶を参照して答えます。

## 4. オフライン（機内モード）での挙動
| できること | できないこと |
|---|---|
| 会話・推論 ✅ | 新しい記憶の同期 ❌（要オンライン） |
| 同期済みの記憶を参照 ✅ | — |

→ 旅行中・電波が無い所でも、**会話と「同期済みメモの参照」は普通に使えます**。

## 5. 最新メモを反映したいとき
1. （母艦PCで）メモをクラウドに同期しておく
2. アプリを**オンライン**にして **🔄** を押す
→ これで最新のメモが端末に入ります。アプリが持つのは「最後に同期した時点」のスナップショットです。

## 6. うまくいかないとき
| 症状 | 対処 |
|---|---|
| 同期で「パスワードが違います」 | Web版のログインPWを再確認（大文字小文字・全角半角） |
| 知ってるはずのメモに答えない | まだ同期していない可能性 → 🔄で同期。もしくはクラウド側がそのメモを未取込 |
| 回答が的外れ・散らかる | 端末内の小型モデルの限界。短く具体的に聞くと安定しやすい |
| 数日後に起動しなくなった | 自作アプリの署名切れ。開発者がMacで再ビルドすると復活 |

## 7. プライバシー
- 会話も記憶も**iPhoneの中だけ**で処理されます（同期の時だけ通信）。
- メモ本文は端末内に保存。パスワードは他人に教えないでください。
